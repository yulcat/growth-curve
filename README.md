# 🌱 성장곡선 (Growth Curve)

WHO 성장 기준표 기반 **쌍둥이 성장 백분위 차트** PWA.

## 특징

- 📊 **WHO 성장 기준표** — 체중, 신장, 두위 백분위 곡선 (0~24개월)
- 👶 **쌍둥이 지원** — 아기별 개별 차트 + 비교 뷰
- 📈 **백분위 계산** — LMS 방법으로 정확한 백분위 산출
- ⚠️ **자동 알림** — 3백분위 미만/97백분위 초과 경고, 쌍둥이 간 체중 차이 모니터링
- 🔄 **교정 월령** — 조산아 보정 지원
- 📥 **둥이로그 연동** — twin-log 앱에서 성장 기록 가져오기
- 🖨️ **인쇄** — 소아과 방문 시 깔끔한 차트 출력
- 📱 **PWA** — 오프라인 지원, 홈 화면 추가 가능
- 🔗 **실시간 동기화** — Socket.io 다기기 공유

## 실행

```bash
npm install
npm start    # http://localhost:3472
```

## 데이터 출처

- [WHO Child Growth Standards](https://www.who.int/tools/child-growth-standards/standards)
- 백분위: P3, P15, P50, P85, P97
